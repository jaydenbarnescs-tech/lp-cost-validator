# lp-cost-validator

月額制LP事業のコスト・利益検証ツール

## URL
https://lp-cost-validator.vercel.app/tools/lp-cost-validator

## 概要
- シナリオ作成タブで月額料金・コスト項目・初期費用・契約期間を設定
- 比較タブで2つのシナリオを並べて検証 (例: Claude Build vs WordPress Build)
- ツールライブラリから Vercel / Sanity / Supabase / Xserver / WordPress などを1クリック追加
- localStorageに自動保存

## Stack
- Next.js 16 + React 19
- Tailwind CSS v4
- TypeScript
